# SDET Learning Journey

This folder contains all learning materials organized by phase.

## Structure

```
learning/
├── phase-a-foundations/     # SDLC, STLC, system flows, Playwright theory, API theory, Git, debugging
├── phase-b-sdlc-app/        # SDLC Simulator App (Project 1)
├── phase-c-automation/      # Automation Framework (Project 3)
└── phase-d-interview/       # Interview preparation, resume, narratives
```

## Current Phase

Check your current phase status with: **"Show current phase"**

## Phase Gates

- **A → B**: Must demonstrate understanding of all Phase A concepts
- **B → C**: App must be complete, documented, architecture understood
- **C → D**: Framework must work, CI/CD green, architecture explainable

## Quick Commands

- `"Start Phase A"` - Begin foundations learning
- `"Check readiness"` - Check current phase gate criteria
- `"Show progress"` - Display overall learning progress
- `"Explain [concept]"` - Get hybrid explanation (flow + structured)

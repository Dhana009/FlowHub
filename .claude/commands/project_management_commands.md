# Project Management Commands

Commands for managing projects and code.

## Architecture & Design
- `@review-architecture [project]` - Review current project architecture
- `@design-architecture [project]` - Help design architecture for project
- `@architecture-approval` - Request architecture approval before coding

## Code Generation
- `@generate-structure [project]` - Generate folder structure for project
- `@code-review [file]` - Review code file for improvements
- `@refactor-suggestion` - Suggest refactoring opportunities

## Documentation
- `@generate-docs [type]` - Generate documentation
  - Types: README, Architecture, API docs, Setup guide
- `@update-docs` - Update existing documentation
- `@doc-checklist` - Check if all required docs are complete

## Testing & Validation
- `@test-strategy [project]` - Define testing strategy for project
- `@validate-implementation` - Validate current implementation
- `@debug-help [issue]` - Help debug current problem
- `@flaky-test-analysis` - Analyze and fix flaky tests

## Project-Specific
- `@sdlc-app-help` - Get help with SDLC Simulator App (Project 1)
- `@mcp-agent-help` - Get help with MCP Hybrid Agent (Project 2)
- `@framework-help` - Get help with Automation Framework (Project 3)

# Resume & Interview Commands

Commands for resume building and interview preparation.

## Resume Management
- `@update-resume` - Help update resume with current skills
- `@resume-bullets [project]` - Generate resume bullets for Project 1, 2, or 3
- `@resume-skills` - List all skills that can go on resume
- `@resume-review` - Review current resume for improvements
- `@resume-versions` - Show different resume versions (A, B, C)

## Project Narratives
- `@project-narrative [project] [length]` - Generate project narrative
  - Examples: `@project-narrative 1 30s`, `@project-narrative 3 3min`, `@project-narrative 2 deep`
- `@narrative-30s [project]` - 30-second pitch for project
- `@narrative-3min [project]` - 3-minute technical overview
- `@narrative-deep [project]` - Deep-dive architecture explanation

## Interview Practice
- `@practice-interview` - Start mock interview session
- `@interview-questions [topic]` - Get interview questions for topic
  - Examples: `@interview-questions Playwright`, `@interview-questions API testing`, `@interview-questions framework design`
- `@behavioral-prep` - Practice behavioral interview questions (STAR method)
- `@technical-prep [area]` - Practice technical questions for specific area

## Interview Scenarios
- `@scenario-question [type]` - Practice scenario-based questions
  - Types: framework design, debugging, system flow, API testing
- `@defend-architecture [project]` - Practice defending project architecture decisions
- `@explain-framework` - Practice explaining automation framework

# Progress & Status Commands

Commands to track learning progress and readiness.

## Readiness Checks
- `@readiness-checklist` - Show SDET readiness checklist (12/12 items)
- `@check-readiness` - Check current phase gate criteria
- `@skill-assessment` - Assess current skill level across all areas

## Progress Tracking
- `@show-progress` - Display overall learning progress across all phases
- `@project-status` - Status of all 3 projects (SDLC App, MCP Agent, Framework)
- `@completed-topics` - List all completed topics in current phase
- `@remaining-topics` - List remaining topics in current phase

## Project Status
- `@project-1-status` - Status of SDLC Simulator App (Phase B)
- `@project-2-status` - Status of MCP Hybrid Memory Agent
- `@project-3-status` - Status of Automation Framework (Phase C)

## Milestone Tracking
- `@milestones` - Show all learning milestones
- `@next-milestone` - What's the next milestone to achieve
- `@phase-gate-status` - Check if ready to move to next phase
